# Releases

Every released version of Obsidian Valet is published on GitHub, with notes and downloadable files.

- **All releases:** [github.com/fgsanz/obsidian-valet/releases](https://github.com/fgsanz/obsidian-valet/releases)
- **Latest release:** [github.com/fgsanz/obsidian-valet/releases/latest](https://github.com/fgsanz/obsidian-valet/releases/latest)

Each release includes downloadable archives — a named source `.zip` (`obsidian-valet-<version>.zip`) plus GitHub's automatic "Source code" `.zip` / `.tar.gz`. To install from a download, unzip it anywhere and run `npm install` inside the extracted folder (see [Add Git to your vault](git-setup) if you also want the git safety features).

For the list of what changed in each version, see the [Changelog](changelog). When **Check for new version** is enabled in Settings, Obsidian Valet checks this releases page at launch and notifies you when a newer version is available.
